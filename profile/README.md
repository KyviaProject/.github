# Kyvia

**An anonymous-first ecosystem. Chat, transact, exist online without ever revealing who you are.**

Kyvia is a suite of open-source tools built for people who refuse to be tracked. No accounts, no phone numbers, no emails, no KYC, no telemetry. Every component is designed from the ground up so that **nobody - not even us - can know who you are**.

## What we're building

💬 **Kyvia Chat** : End-to-end encrypted messaging with complete anonymity. No phone number, no email, no identity required. 

🔐 **Kyvia Wallet** : A hardware wallet using a J2A040 JCOP smart card as a secure element. Private keys are generated and stored on the card — they never leave it. No cloud, no server, no third party.

💳 **Java Card Applet** : Custom applet deployed on the smart card, handling key management, PIN protection, and on-card signing via SCP02 secure channel. 

🌐 **Kyvia Website** : Project hub and documentation.

## Core principles

- **100% anonymous** : No accounts, no registration, no identity. You are a key, not a name.
- **No tracking, no telemetry** : None of our software ever contacts a server to report on you. Zero analytics. Zero data collection.
- **End-to-end encrypted** : Messages and transactions are encrypted. No one in between can read them.
- **Keys never leave the card** : Cryptographic operations happen inside the secure element.
- **Fully open source** : Every line of code is auditable. Don't trust — verify.
- **Offline-capable** : The wallet works without an internet connection. Your keys are yours even if the network goes down.
